This was a class project for MSCS 6050: Elemnents of Software Development at Marquette University. The idea of building a Smart Web Crawler was originally concieved by the staff of Direct Supply. The original goal was to be able to use the R programming language to scrape the web given two or more topics and be able to connect them through a topic modeling algorithm. Currently, we are hitting directly upon the Wikipedia website to return a list of topics relevant to the initial input words.
